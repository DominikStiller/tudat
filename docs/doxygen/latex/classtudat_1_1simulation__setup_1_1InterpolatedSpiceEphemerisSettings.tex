\doxysubsection{Detailed Description}
@get\+\_\+docstring(Interpolated\+Spice\+Ephemeris\+Settings.\+\_\+\+\_\+docstring\+\_\+\+\_\+) 

\mbox{\hyperlink{classtudat_1_1simulation__setup_1_1EphemerisSettings}{Ephemeris\+Settings}} derived class for defining settings of a ephemeris interpolated from Spice data.

\mbox{\hyperlink{classtudat_1_1simulation__setup_1_1EphemerisSettings}{Ephemeris\+Settings}} derived class for defining settings of a ephemeris interpolated from Spice data. Retrieving a state from Spice can be very computationally intensive. Using the settings of this class, ephemeris data for a body is pre-\/computed using a limited number of calls to Spice, which is then used to create an interpolator (6th order Lagrange). For many numerical integration scenarios, this approach may be faster than using \mbox{\hyperlink{classtudat_1_1simulation__setup_1_1DirectSpiceEphemerisSettings}{Direct\+Spice\+Ephemeris\+Settings}}, with negligible influence on accuracy.
