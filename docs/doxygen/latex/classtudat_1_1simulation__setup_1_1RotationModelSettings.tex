\doxysubsection{Detailed Description}
Class for providing settings for rotation model. 

Class for providing settings for automatic rotation model creation. This class is a functional (base) class for settings of rotation models that require no information in addition to their type. Rotation model classes defining requiring additional information must be created using an object derived from this class. @get\+\_\+docstring(Rotation\+Model\+Settings.\+\_\+\+\_\+docstring\+\_\+\+\_\+)
